One-Click Production

The included server.js implements all five endpoints as a stateless Express service. Drop the folder into Railway, set your environment variables, and you have a fully operational key verification API.

  1. 1Fork or upload the project files to a GitHub repo.
  2. 2Create a new project on Railway and select "Deploy from GitHub repo".
  3. 3Set KEY_PREFIX and WORKINK_SLUG in Railway variables.
  4. 4Railway auto-detects Node.js and runs npm start. Your API is live.
